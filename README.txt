Объединение последовательностей одинаковых идущих подряд строк в файле в одну:
  Флаг ​"-i" ​означает, что при сравнении строк не учитывается регистр символов.
  Флаг ​"-u" означает, что будут выведены в качестве результата только уникальные​ строки.
  Флаг ​"-r" означает, что перед каждой строкой вывода будут выведены количество строк, которые были заменены данной
   (т.е. если во входных данных было 2 одинаковые строки, в выходных данных будет одна с префиксом “2”).
  Флаг ​"-s ​N" означает, что при сравнении строк игнорируются первые N символов каждой строки. Выводит первую строку.
  Флаг "-o ofile" задаёт имя выходного файла. Если параметр отсутствует, выводит результаты на консоль.
  "file" - ​задаёт имя входного файла. Если параметр отсутствует, считывает текст с консоли.

Command line: uniq [-i] [-u] [-c] [-s N] [-o ofile] [file]
